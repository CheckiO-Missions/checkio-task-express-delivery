Unsere drei Roboter haben auf der Insel ein paar mysteriöse Kisten gefunden. Nach einiger Untersuchung entdeckte Nicola, dass diese kisten eine interessante Eigenschaft haben. Wenn Sie etwas in eine dieser Boxen legen, können Sie es aus jeder anderen Box wieder herausholen. Stephan meint, dass dies eine schnelle Lieferung von Fracht über die Insel ermöglicht und die Lasten doppelt so schnell transportiert werden können. Stephan kann die Fracht in eine Kiste legen und sie später am Lieferort abholen. Auf der Karte gibt es Wasserzellen, die Stephan nicht passieren kann, aber sonst machen diese Kisten seine Aufgabe sehr viel erleichtern.

Die Karte für die Lieferung wird als eine Reihe von Strings dargestellt, wobei:

  • "W" ist ein Wasser (geschlossene Zelle)
  • "B" ist eine Schachtel
  • "E" ist ein Zielpunkt.
  • "S" ist ein Startpunkt.
  • "." ist eine leere Zelle.

Stephan bewegt sich zwischen benachbarten Zellen in zwei Minuten, wenn er eine Ladung trägt. Ohne Handgepäck benötigt er nur eine Minute. Das Be- und Entladen der Fracht in die (und aus der) Box dauert eine Minute. Sie sollten den schnellsten Weg für die Lieferung der Fracht finden (minimale Zeit).

Die Route ist eine Zeichenkette, bei der jeder Buchstabe für eine Aktion steht.

  • "U" - Nach oben (Norden)
  • "D" -- Nach unten (Süden)
  • "L" -- Links (Westen)
  • "R" -- Rechts (Osten)
  • "B - Einladen oder Entladen einer Kiste (aus einer Kiste).

Eingabe: Eine Karte zur Übergabe als Liste von Strings.

Ausgabe: Die schnellste Route als String.

Beispiel: 

checkio(["S...","....","B.WB","..WE"]) #RRRDDD
checkio(["S...","....","B..B","..WE"]) #DDBRRRBD

Wie es verwendet wird: Diese Aufgabe ähnelt dem "Offenen Labyrinth", aber hier haben wir verschiedene Kosten für die Züge. Anhand dieser Aufgabe lernen Sie Algorithmen kennen, die bei der Wegfindung und der Durchquerung von Graphen verwendet werden. Ein Beispiel: In einigen Strategie- oder Rollenspielen kann sich eine Einheit mit unterschiedlicher Geschwindigkeit auf verschiedenen Terrains bewegen.

Vorbedingung: 0 < Zeilen < 10
0 < Spalten < 10
∀ x,y ∈ Koordinaten: 0 < x,y < 10
